Validating Web Forms

First name must be alphanumeric and contain 3-16 charecters

Last name must be alphanumeric and contain 3-16 charecters

Email must be valid email address Eg: example@example.com

Password must be alphanumeric with special charecters like !,@,#,$,Etc and minimum 8 charecters

Telephone must be valid number Eg: 11 Digits and 333-333-3334

Bio may contain alphanumeric character, comma, period, apostrophe, double-quote, space, or hyphen and minimum 10 charecters.
